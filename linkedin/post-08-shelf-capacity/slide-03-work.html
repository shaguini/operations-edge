What We Did 3 / 5

5 Whys. One location.
One structural fix.

1
Why: wrong item picked

Products were stored adjacently with poor visual separation — overcrowded pick face.

2
Why: pick face was overcrowded

Replenishment was filling the location to system-defined capacity.

3
Why: system capacity exceeded physical capacity

WMS capacity figure exceeded what the shelf physically held. The difference was never flagged.

4
Why: no process to audit system vs physical capacity

Capacity was set at system go-live and never reviewed. No trigger for a re-audit.

5
Root cause: no slotting/capacity review process existed

Fix: physical capacity audit → update WMS → standardise replenishment fill limits → schedule recurring reviews.
