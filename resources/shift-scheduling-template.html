Weekly Shift Schedule Template

Assign shifts, track weekly hours, and flag overtime — all in your browser. No sign-up required. Print to PDF when you're done.

How to use this template: Type a shift code into each cell — E (Early), L (Late), N (Night), D (Day), H (Holiday), X (Off) — or any custom code your operation uses. Hours are estimated automatically based on your shift definitions below. Weekly totals flag red when they exceed the overtime threshold (default: 48 hrs, the UK Working Time Regulations limit).

Shift durations used for hour calculations: E = 8 hrs  |  L = 8 hrs  |  N = 10 hrs  |  D = 8 hrs  |  H = 0 hrs (leave)  |  X = 0 hrs. You can adjust these in the code or override manually.
